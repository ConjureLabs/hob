<kbd>hob</kbd>

![](https://media0.giphy.com/media/FD3of7ehZt4li/giphy.gif)

Hob (short for Hobgoblin) does chores in the background, helping your app run smoothly. Usually.

### Setup

```sh
npm install --save @conjurelabs/hob
```

or

```sh
yarn add @conjurelabs/hob
```

### Building

`hob build`
